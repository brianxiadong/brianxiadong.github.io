这是你的第一篇博客文章！

Jekyll 是一个静态网站生成器，非常适合搭建个人博客。你可以：

- 使用 Markdown 编写文章
- 自定义主题和样式
- 部署到 GitHub Pages

## 如何添加新文章

在 `_posts` 目录下创建新的 Markdown 文件，文件名格式为：

```
YYYY-MM-DD-title.md
```

例如：`2026-01-28-my-first-post.md`
